## ✨ Features

- 🏠 **Hero Section** — Animated landing with CTA buttons (WhatsApp & Call)
- 🔧 **Services Grid** — 8 repair services with scroll-reveal animations
- 🏷️ **Brand Support** — Displays all supported laptop brands
- ⭐ **Testimonials** — Customer reviews with hover effects
- 📍 **Location & Map** — Embedded Google Maps with contact details
- 📱 **Fully Responsive** — Mobile-first design, works on all screen sizes
- 💬 **WhatsApp Float Button** — Fixed floating button for quick contact
- 🎨 **Smooth Animations** — IntersectionObserver-based reveal animations
- 🔒 **Privacy Policy & Terms** pages included

## 🚀 Installation & Setup

### Prerequisites
- Node.js `v18+`
- npm or yarn

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/your-username/tech-team-store.git

# 2. Navigate into the project
cd tech-team-store

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will run at **http://localhost:5173**

### Build for Production

```bash
npm run build
```

Output will be in the `/dist` folder — ready to deploy on Vercel, Netlify, or any static host.
